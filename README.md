     ```bash
     npm install
     ```

   - For the backend (Node.js):
     ```bash
     cd backend
     npm install
     ```

   Make sure to configure the database connection in the `.env` file and run the necessary migrations to set up the database schema.

   - Start the frontend:
     ```bash
     ng serve
     ```
